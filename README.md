Date Format Conversion based on input
======================

Accepts string in different format with various kind of separators.


### Input formats it accepts:
(Regardless of separator)
* mm-dd
* dd-mm
* MMM-dd
* dd-MMM
* mm-dd-yyyyy
* dd-mm-yyyy
* yyyy-MM-dd
* MMM-dd-yyyy
* dd-MMM-yyyy

### Separatos it can accepts:
- Comma (,)
- Dot (.)
- Dash (-)
- Slash (\, /)
- anything else
